Découverte HTML & CSS

En effet, nous avons pu constater que les langages HTML et CSS étaient indispensables pour créer et mettre en page un site Internet. L'utilisation de ces deux langages est l'une des étapes les plus importantes dans la conception d'un site Internet.

  • Le HTML (HyperTextMarkup Languages) est un langage utilisé pour mettre en page et pour structurer un site Internet ou un blog. C'est également avec ce langage que nous pouvons faire apparaitre du texte dans nos sites et dans nos blogs.
  • Le CSS est un langage utilisé pour structurer un code. On l'utilise pour, par exemple, mettre en couleur ou en gras un texte, mettre des effets sur un texte (italique,gras,etc...). On utilise également ce langage pour gérer la taille d'une image ou pour la changer de position. Ce langages est extremement utile en programmation !
